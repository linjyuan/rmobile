# Rmobile

## 特性

- 🚀 UI 的快速实现。
- 🎉 实现一次性全部赋值。
- 🍁 表单提交取值。
- 💄 融合多类型组件表单。
- 🌈 支持动态表单。
- 🐠 公司内部数十个项目中得到锤炼，不断优化完善。

## 快速上手

```bash
npm install Rmobile

or

yarn add Rmobile

or

pnpm add Rmobile
```
