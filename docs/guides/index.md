# 指南

## 安装
```
$ npm install --save antd-mobile
# or
$ yarn add antd-mobile
# or
$ pnpm add antd-mobile
# or
$ bun add antd-mobile
```

## 引入
直接引入组件即可，antd-mobile 会自动为你加载 css 样式文件：
```
import { Button } from 'Rmobile';
```
